# Magic 8-Ball Game

A simple command-line implementation of the classic Magic 8-Ball fortune-telling toy.

## Description

This Python program simulates the Magic 8-Ball experience, where users can ask yes-or-no questions and receive mystical answers. The application randomly selects responses from a predefined list of traditional Magic 8-Ball answers.

## How to Play

1. Run the program:

   ```bash
   python magic8ball.py
   ```

2. Enter your yes-or-no question when prompted
3. Receive the Magic 8-Ball's mystical answer

## Features

- Simple command-line interface
- Classic Magic 8-Ball responses
- Random answer generation

## Implementation Details

The program is implemented in [magic8ball.py](magic8ball.py) and uses Python's built-in `random` module to select from 9 possible answers:

- Affirmative answers (e.g., "Yes - definitely", "It is decidedly so")
- Non-committal answers (e.g., "Reply hazy, try again", "Ask again later")
- Negative answers (e.g., "My sources say no", "Very doubtful")

## Requirements

- Python 3.x
- No additional libraries required

## Example Usage

```
$ python magic8ball.py
Ask a yes or no question please 
Question: Will I win the lottery?
Magic 8-Ball: Very doubtful.
```
